## Get Started in 3 Steps

### Step 1: Choose Your Model

### Step 2: Add Your API Key

#### For Claude (Anthropic):
1. Get your free key at: https://console.anthropic.com
2. Copy the API key (starts with `sk-ant-...`)
3. Tap "Set Claude API Key" in the app
4. Paste and save

#### For Gemini (Google):
1. Get your free key at: https://aistudio.google.com/apikey
2. Copy the API key (starts with `AIza...`)
3. Tap "Set Gemini API Key" in the app
4. Paste and save

### Step 3: Start Using AI Extraction
1. Go to "Tell a Story"
2. Write about an experience with friends
3. Tap "Save & Extract"
4. Watch the AI extract relationships!

## Common Questions

**Q: Can I use both models?**
A: Yes! Set up keys for both and switch between them anytime.

**Q: Which model should I use?**
A: 
- Claude: Better for complex stories with subtle details
- Gemini: Faster and more cost-effective for simple stories

**Q: Do I need to pay?**
A: Both offer free tiers:
- Anthropic: Free credits on signup
- Google: Free tier with generous limits

**Q: Are my API keys safe?**
A: Yes! Keys are stored locally on your device and only sent to the respective AI provider.

**Q: What if I don't have a key?**
A: The app will still save your story. You just won't get automatic AI extraction until you add a key.
